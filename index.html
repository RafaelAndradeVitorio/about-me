O que esse projeto prova

O Nexo reúne frontend, UX, métricas, segurança e regras de negócio. A interface foi pensada para transformar informação operacional em algo claro, visual e acionável para escolas.

250+ alunos gerenciados
R$70k+ em cobranças acompanhadas
RLS isolamento por tenant
PIX + Recorrência Mercado Pago
  • ✓Arquitetura multi-tenant com organization_id e Row Level Security no Supabase/Postgres.
  • ✓Integração com Mercado Pago para pagamento avulso, PIX, assinaturas recorrentes e cobrança mensal.
  • ✓Webhooks idempotentes e reconciliação de cobranças para reduzir falsos inadimplentes.
  • ✓KPIs financeiros para esperado, carteira e inadimplência com maior controle operacional.
  • ✓Integração com WhatsApp via Evolution API para mensagens, broadcasts e alertas.
  • ✓Desenvolvimento assistido por IA com revisão técnica, testes e documentação para acelerar sem abrir mão de qualidade.
